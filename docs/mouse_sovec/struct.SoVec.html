SoVec stands for Small optimized Vector .

SoVec behaves like std::collections::Vec, however, it will not allocate heap memory if the requested memory size is small enough. Instead of heap, it uses itself as a buffer then.

To avoid allocating as much as possible, the performance is better than that of std::collections::Vec .

Some unsafe methods behavior is different from the same name method of std::collections::Vec ; otherwise, (i.e. the all safe methods) behaves like that of std::collections::Vec .

Warnings
